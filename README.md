## About

SEObolt is a full-featured SEO plugin for WordPress: on-page analysis, schema markup, XML sitemaps, redirects, 404 monitoring, AI content helpers, and a GEO Score that measures how well your content is optimized for AI-driven search engines (ChatGPT, Claude, Perplexity, Google AI Overviews). Everything runs inside WordPress — no external services required for the core feature set.

The free version is available on WordPress.org and covers on-page analysis, schema generation, sitemaps, and redirects. The Pro version at royalplugins.com adds AI content helpers, advanced schema templates, import from other SEO plugins, GSC and GA4 analytics inside WordPress, and full GEO Score reporting. Both versions integrate cleanly with Gutenberg, Elementor, Divi, Beaver Builder, WPBakery, Oxygen, Bricks, Avada, and Thrive Architect.

SEObolt is part of a broader SEO toolkit that includes the [SEObolt SaaS dashboard](https://seobolt.io) for rank tracking plus AI search visibility, a free [Chrome extension](https://github.com/royalplugins/seobolt-chrome-extension) for one-click on-page audits, and free SEO web tools (SERP Preview, Schema Validator, Meta Tag Checker). The WordPress plugin is the local-first entry point for teams that want their SEO work to live inside the CMS rather than in a separate subscription.

## Features

- 57-point on-page SEO analysis with real-time scoring as you edit
- 63 schema templates including Article, FAQ, HowTo, Recipe, Event, LocalBusiness, Product
- GEO Score — 37-factor AI citation readiness (content structure, evidence, E-E-A-T, freshness)
- Content AI with OpenAI, Anthropic, DeepSeek, or Gemini for titles, meta descriptions, and keyword suggestions
- XML sitemaps — post, page, image, video, news variants with auto-ping
- Redirect manager — 301, 302, 307, 308 with exact, prefix, and regex matching
- 404 monitor with one-click redirect creation
- `llms.txt` dynamic editor served at your site root
- Google Search Console + Google Analytics 4 OAuth integration
- Import from Yoast, Rank Math, All in One SEO, SEOPress
- 9 page-builder integrations (Gutenberg, Elementor, Divi, Beaver Builder, WPBakery, Oxygen, Bricks, Avada, Thrive Architect)
